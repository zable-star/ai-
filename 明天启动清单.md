## 🛠️ 技术栈建议（以 AI 角色扮演为例）

### 前端
- React / Vue.js
- Ant Design / Material-UI
- Web Audio API（语音）
- WebSocket（实时通信）

### 后端
- Node.js (Express/Koa) 或 Python (FastAPI)
- LLM API：OpenAI / 通义千问 / 文心一言
- 语音识别：讯飞/腾讯云/Azure Speech
- 语音合成：同上

### 存储
- 七牛云对象存储（聊天记录、角色配置）
- 七牛云 CDN（静态资源加速）

### 部署
- 前端：Vercel / 七牛云静态托管
- 后端：云服务器 / Serverless

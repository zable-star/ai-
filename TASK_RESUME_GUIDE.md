## 🐛 可能遇到的问题

### 问题 1: 听不到语音
**原因**: 浏览器可能没有加载语音列表  
**解决**: 
1. 打开控制台
2. 运行 `window.speechSynthesis.getVoices()`
3. 检查是否有中文语音
4. 刷新页面重试

### 问题 2: 新对象不显示
**原因**: JavaScript 语法错误或浏览器缓存  
**解决**:
1. 打开控制台查看错误
2. 强制刷新 (Ctrl+F5)
3. 清除浏览器缓存

### 问题 3: AI 不理解新对象
**原因**: 需要配置 AI 模型  
**解决**:
1. 点击"模型配置"
2. 选择 AI 提供商并输入 API Key
3. 或使用本地解析（限制更多）

### 问题 4: 后端连接失败
**原因**: 后端服务未启动  
**解决**:
```bash
cd backend
npm run dev
```
